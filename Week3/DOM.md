### 💬 문서 객체 모델, DOM

- 태그를 자바스크립트에서 사용할 수 있는 객체로 만든 것.
- 웹 문서를 메모리로 읽어들여 트리 구조로 변환한다.
- 문서의 각종 요소는 요소 노드(**`element node`**), 텍스트는 텍스트 요소(**`text node`**)로 변환한다.
- 변환된 트리 구조를 이용하면 자바스크립트로 웹 문서를 조작할 수 있다.

  ![Image](https://github.com/user-attachments/assets/b3deacea-16b6-4477-8b48-462e83e56a09)

- **`정적 생성`**
  - 웹 페이지를 처음 실행할 때 HTML 태그로 적힌 문서 객체를 생성하는 것.
- **`동적 생성`**
  - 웹 페이지를 실행 중에 자바스크립트를 사용해 문서 객체를 생성하는 것.
  - 자바스크립트는 웹 페이지의 텍스트나 스타일을 동적으로 변경할 수 있다.

| **노드의 종류** | **설명**                             |
| --------------- | ------------------------------------ |
| 요소 노드       | <p class=”container”>안녕하세요.</p> |
| 속성 노드       | class=”container”                    |
| 텍스트 노드     | 안녕하세요                           |

> **동적 표시 요소를 추가할 때**

| **메소드**             | **설명**                     |
| ---------------------- | ---------------------------- |
| **`createElement()`**  | 요소 노드를 생성             |
| **`createTextNode()`** | 텍스트 노드를 생성           |
| **`appendChild()`**    | 요소 노드를 Body 객체에 추가 |

### 🎀 이벤트 처리

- **`이벤트`** : 웹 문서에서 사용자가 취하는 모든 행위
- **`이벤트 리스너`** : 이벤트가 발생했을 때 실행되는 코드
- 웹사이트는 클릭, 탭, 스크롤 등 다양한 이벤트가 존재하는데 이러한 이벤트 각각에 따른 이벤트 처리가 가능하다.
  - **`addEventListener()`** 를 사용하여 제어한다!
